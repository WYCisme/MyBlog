### 从仓库拉去源代码之后
进入该文件并在当前路径下打开git bash here，依次运行如下命令来进行搭建hexo环境：
```sh
npm install hexo-cli -g
npm install hexo --save
```
### 创建一篇新post

```sh
hexo new 文章名称
```

### 生成并更新hexo本地文件

```sh
hexo g
```

### 部署到gitee page

```sh
hexo d
```

生成并部署

```sh
hexo g -d
```

### 最后执行一键更新

```
峡州仙士GiteePage一键自动部署.exe - 快捷方式
```
